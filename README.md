## Getting Started

### Prerequisites

-   Java 8 or higher
-   Python 3.8 or higher
-   Git 2.25.1 or higher
-   Gradle 6.8.3 or higher
-   IDE (IntelliJ IDEA, VSCode, etc.)

### Installing

1. Clone this repository
2. Open the project in your IDE
3. Run the project
4. Choose the module you want to run
